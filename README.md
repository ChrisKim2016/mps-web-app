## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/ChrisKim2016/mps-web-app.git
cd mps-web-app
```

### 2. Run Locally

You can use any local server to run the application:

#### Using Python 3:
```bash
python -m http.server 8000
```

#### Using Python 2:
```bash
python -m SimpleHTTPServer 8000
```

#### Using Node.js (with http-server):
```bash
npx http-server
```

### 3. Open in Browser

Navigate to `http://localhost:8000` in your browser.

## Customization

### Colors
Edit the CSS variables in `styles.css` (`:root` section):

```css
:root {
    --primary-color: #2563eb;      /* Change primary blue */
    --secondary-color: #1e40af;    /* Change secondary blue */
    --text-color: #1f2937;         /* Change text color */
    --light-bg: #f3f4f6;           /* Change light background */
}
```

### Content
- Edit `index.html` to change text, headings, and content
- Modify section titles and descriptions
- Update the footer with your information

### Styling
- Adjust padding, margins, and sizing in `styles.css`
- Modify font sizes and families
- Change animation speeds and effects

### Functionality
- Update form handling in `script.js`
- Add new interactivity as needed
- Modify scroll behavior and animations
